# Ứng dụng đặt đồ ăn nhanh trên Android

Lập trình toàn bộ giao diện và xử lý logic, thiết kế cấu trúc cơ sở dữ liệu, phân tích thiết kế hệ thống.

- Java
- XML
- SQLite
- Android Studio

### Quản trị viên:
